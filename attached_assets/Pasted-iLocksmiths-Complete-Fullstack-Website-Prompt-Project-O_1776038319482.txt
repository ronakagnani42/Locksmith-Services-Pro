iLocksmiths - Complete Fullstack Website Prompt
Project Overview
Build a professional, conversion-optimized locksmith website for iLocksmiths, a 24/7 emergency locksmith service operating in London, UK. The website should have a dark minimalist design inspired by CRED, with cinematic video backgrounds, sophisticated animations, and a focus on trust and conversion.

Key Implementation Notes
1. Video Background
Use HTML5 <video> element
Attributes: autoPlay, muted, loop, playsInline
Desktop: Show video
Mobile: hidden md:block (hide on mobile)
Overlay: bg-gradient-to-r from-black/70 via-black/50 to-transparent
2. Framer Motion Usage
<motion.div
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true, margin: "-100px" }}
  variants={fadeInUp}
  transition={{ duration: 0.6 }}
>
  Content
</motion.div>
3. WhatsApp Integration
const handleWhatsApp = () => {
  const message = encodeURIComponent("Hi, I need help with...");
  window.open(`https://wa.me/${businessInfo.whatsapp}?text=${message}`, '_blank');
};
4. Responsive Video
{/* Desktop/Tablet - Video */}

{/* Mobile - Static Background */}

Final Notes
This website should feel premium, trustworthy, and professional. The dark minimalist design combined with cinematic video and sophisticated animations creates a high-end feel that builds immediate trust. Every element should serve the goal of converting visitors into customers through clear messaging, easy contact methods, and a seamless user experience.
